  • cachedRunQuery ∷ ∀ a. Qry a → STM m (Either PastHorizonException a)

    Run the specified query

    If the query fails with a PastHorizonException, it will update its internal state (compute a new summary) and try again. If that still fails, the PastHorizonException is returned.

    See also cachedRunQueryThrow.
