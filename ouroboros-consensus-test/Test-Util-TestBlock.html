  • tbHash ∷ TestHash
     
  • tbSlot ∷ SlotNo

    We store a separate SlotNo, as slots can have gaps between them, unlike block numbers.

    Note that when generating a TestBlock, you must make sure that blocks with the same TestHash have the same slot number.

  • tbValid ∷ Bool

    Note that when generating a TestBlock, you must make sure that blocks with the same TestHash have the same value for tbValid.
